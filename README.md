# Projet Dashboard 
 ## Description

Ce projet consiste à réaliser un dashboard moderne en utilisant le style Glassmorphism avec un background animé afin d’apporter plus de dynamisme à l’interface,
Le projet a été réalisé avec un framework CSS afin de créer une interface responsive, élégante et moderne.



##  Objectifs
Découvrir et appliquer le design Glassmorphism
Créer une interface utilisateur moderne et responsive
Intégrer un background animé dans une page web
Utiliser les classes d’un framework CSS
Adapter du code externe à son propre projet


## Technologies Utilisées
HTML5
CSS3
Bootstrap 5
Bootstrap Icons
